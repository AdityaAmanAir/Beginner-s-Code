Smooth Shutter Reveal

Scroll down to see the shutter effect in action

This effect simulates a camera shutter opening and closing. As you scroll:

  • The top gradient moves up to reveal the image
  • The bottom gradient moves up to hide the image again
  • Use the controls to customize the gradient colors

How It Works

The shutter effect is created using two gradient layers positioned above and below the main image. As you scroll, JavaScript calculates your scroll position and moves these gradients accordingly.

The top gradient moves upward to reveal the image, and then the bottom gradient moves upward to cover it again, creating a smooth transition that mimics a camera shutter.

Try customizing the colors using the controls on the right to create your own unique effect!
